# Public Brand Knowledge Sources

This file records the public reference layer used by the internal consulting app. It is not a full product catalog, price list, inventory system, or a substitute for checking the exact SKU available at DongDo Optic.

## How The App Uses This Data

- Public brand information adds talking points and confidence signals for staff.
- The app should not promise an exact lens/frame feature unless that SKU is mapped in the store catalog.
- Public data can guide recommendation ranking, but final selling advice still needs prescription, fit, budget, stock, and staff confirmation.

## Brands

### Fano

Sources:
- https://www.fano.vn/
- https://www.fano.vn/blue%20ai-trong-kinh-chong-anh-sang-xanh-tot-nhat-c-25.html

Usable factors:
- Budget-friendly baseline for daily consulting.
- Public references to Blue AI / UV420 / blue-light positioning.
- Public frame/material references such as TR90 can support lightweight/durable advice.

Guardrail:
- Verify exact index, coating, frame material, origin, and stock before quoting to a customer.

### Carl Zeiss

Sources:
- https://www.zeiss.com/vision-care/us/need-new-eyeglasses/clearview-single-vision-lenses.html
- https://www.zeiss.com/vision-care/us/eye-care-professionals/lenses/coatings-technologies/premium-coatings-portfolio.html

Usable factors:
- ClearView: thin/light single vision positioning, broad clarity, UV protection up to 400 nm.
- DuraVision: anti-reflective coating, scratch resistance, easier cleaning, front/back UV protection messaging.
- Good for premium, all-day wear, aesthetics, driving, and higher prescription discussions.

Guardrail:
- For heavily personalized needs, check whether the store sells made-to-order or customized ZEISS options.

### Essilor

Sources:
- https://www.essilor.com/us-en/products/
- https://www.essilor.com/us-en/products/crizal/

Usable factors:
- Crizal as a public anti-reflective/protective coating family.
- Eyezen and digital-device use cases from public product positioning.
- Broad ecosystem for single vision, progressive, UV, and blue-violet filtering discussions.

Guardrail:
- Do not merge all Essilor features into one product. Exact line, coating, and index must be mapped.

### Essilor Element

Source:
- https://visionplusmag.com/business/essilor-launches-elements-blue-light-protection-lenses/

Usable factors:
- Public blue-light, UV, anti-reflection, smudge-resistant, and scratch-resistant positioning.
- Useful as an accessible upgrade from basic lenses for screen-heavy customers.

Guardrail:
- Present as value-oriented, not as a premium personalized product line.

### Gọng kính 101

Sources:
- https://101eyewear.com/
- https://101eyewear.com/thuong-hieu/
- https://101eyewear.com/en/

Usable factors:
- Public brand ecosystem with multiple style segments.
- Minimal/lightweight: Silhouette, Flair.
- Luxury/executive/fashion: Chopard, S.T. Dupont, Dolce&Gabbana, Porsche Design.
- Active/youthful: Fila, New Balance, and similar segments if stocked.

Guardrail:
- 101 is a brand/distribution reference layer. The app still needs DongDo's real SKU data: size, bridge, temple length, material, color, price, and stock.
